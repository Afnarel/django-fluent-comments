Instead, define DATETIME_FORMAT in a locale file. Requires setting: FORMAT_MODULE_PATH = 'settings.locale' Then create 'settings/locale/XY/formats.py' with: DATETIME_FORMAT = '...' This should give you consistent dates across all views. {% endcomment %} {% load i18n %}
